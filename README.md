## 简介

## 内容概述

本书全面覆盖了在车辆控制系统模型的开发过程中使用到的车辆控制系统及动力学模型。具体内容包括：

- 巡航控制
- 自适应巡航控制
- ABS（防抱死制动系统）
- 自动车道保持
- 自动高速系统
- 航向角稳定控制
- 引擎控制
- 被动、主动和半主动悬挂系统
- 轮胎-路面摩擦系数估计
- 防倾翻系统
- 混合电力车辆
